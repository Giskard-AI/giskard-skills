# Getting Started

1. Create an organization in the Acme Analytics console.
2. Connect a read-only database (Postgres, MySQL, or BigQuery).
3. Run the schema sync wizard.
4. Ask questions in natural language; the agent generates SQL against your connected database.

Supported database engines: Postgres 14+, MySQL 8+, BigQuery. SQLite is supported for demos only.

Support hours: Monday–Friday, 9am–6pm UTC.
