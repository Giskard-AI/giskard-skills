# giskard-skills
Agent skills to streamline adoption and usage of Giskard products

## Available Skills

| Skill | Path | When to Use |
|-------|------|-------------|
| scenario-generator | `oss/checks/scenario-generator` | Use when a user describes their AI agent and wants to create adversarial test scenarios, red-team their AI, generate evaluation suites, or build checks using the `giskard.checks` library. Triggers on phrases like "create scenarios", "test my agent", "evaluate my chatbot", "red-team my AI", or "generate checks". |
